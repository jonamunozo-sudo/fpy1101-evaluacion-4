# Proyecto: Sistema de Gestión de Biblioteca
Este proyecto corresponde a la Evaluacion 4 de Fundamentos de Programacion.

El programa permite administrar una biblioteca mediante un menu interactivo desarrollado en Python, el cual incluye las siguientes funcionalidades:

Agregar libro: Permite registrar nuevos textos en el sistema.

Buscar libro: Localiza un libro especifico por su titulo.

Eliminar libro: Remueve un registro del sistema.

Actualizar disponibilidad: Ajusta de manera automatica el estado del libro segun su stock.

Mostrar libros: Despliega la lista completa de libros con sus detalles.

Salir del sistema: Finaliza la ejecucion del programa de forma limpia.

Validaciones del Sistema
Para garantizar la integridad de los datos, el programa valida estrictamente que:

El titulo no este vacio ni contenga solo espacios en blanco.

Las copias sean un numero entero mayor o igual a 0.

El período de préstamo sea un numero entero mayor que 0.
